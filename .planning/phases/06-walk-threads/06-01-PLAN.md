  <name>Task 1: Rewrite useRadarLogic hook from scratch</name>
  <files>aini-inu-frontend/src/hooks/useRadarLogic.ts</files>
  <action>
Delete the entire existing useRadarLogic hook and rewrite from scratch. The new hook must:

**State:**
- `activeTab`: 'FIND' | 'RECRUIT' | 'EMERGENCY' (default 'FIND')
- `coordinates`: [number, number] -- GPS-acquired or Seoul City Hall fallback
- `gpsLoading`: boolean -- true until GPS acquired
- `threadList`: ThreadSummaryResponse[] -- sidebar list data (accumulated from paginated fetches)
- `threadListPage`: number -- current page for load-more
- `threadListHasNext`: boolean -- from SliceResponse
- `mapMarkers`: ThreadMapResponse[] -- coordinate-only map data
- `hotspots`: ThreadHotspotResponse[] -- hotspot aggregation data
- `selectedThread`: ThreadResponse | null -- detail for popup (fetched via getThread on selection)
- `myPets`: PetResponse[] -- user's pets for create/apply
- `isLoading`: boolean -- initial data load
- `isRefreshing`: boolean -- manual re-search in progress
- `currentTime`: Date -- updated by local timer for expiry display
- `editingThreadId`: number | null -- thread being edited (for RECRUIT tab pre-fill)

**GPS acquisition (on mount, once):**
- Call `navigator.geolocation.getCurrentPosition()` with `{ timeout: 10000, enableHighAccuracy: false }`
- On success: round lat/lng to 6 decimal places via `Number(val.toFixed(6))`, set `coordinates`
- On failure (permission denied, timeout, unsupported): set `coordinates` to `[37.566295, 126.977945]` (Seoul City Hall)
- Also update `useConfigStore.setCoordinates()` with the acquired coordinates
- Set `gpsLoading = false` after either success or failure

**Data fetch (after GPS ready):**
- useEffect watching `coordinates` -- when coordinates change and gpsLoading is false, call `fetchThreadData(coordinates)`
- `fetchThreadData`: fires `Promise.all([getThreads({ page: 0, size: 20 }), getThreadMap({ latitude, longitude, radius: 5 }), getHotspots()])`
- On success: set `threadList` from `content`, set `threadListPage` to 0, `threadListHasNext` from `hasNext`, set `mapMarkers`, set `hotspots`
- On error: `toast.error('스레드를 불러오는데 실패했습니다.')`, set `isLoading = false`
- Also fetch `getMyPets()` on mount (for pet-owner check and create/apply forms)

**Load-more (sidebar pagination):**
- `loadMore()`: fetch `getThreads({ page: threadListPage + 1, size: 20 })`, append `content` to existing `threadList`, update `threadListPage` and `threadListHasNext`

**Manual re-search:**
- `handleRefresh()`: sets `isRefreshing = true`, calls `fetchThreadData(coordinates)`, sets `isRefreshing = false`. This is the ONLY way to refetch data (DEC-029).

**Local expiry timer:**
- `setInterval(() => setCurrentTime(new Date()), 60000)` -- every 60 seconds, display-only, NOT a data fetch
- Clean up on unmount

**Thread expiry check:**
- `isExpired(startTime: string): boolean` -- parse ISO datetime string `startTime`, check if `Date.now() - Date.parse(startTime) >= 60 * 60 * 1000`
- Export as utility for components to filter/display expired threads
- Note: startTime is ISO 8601 format (e.g. '2026-03-06T14:00:00'), NOT 'HH:mm' -- do NOT split on ':'

**Thread selection:**
- `selectThread(threadId: number)`: calls `getThread(threadId)` and sets `selectedThread` with the full detail response. Used when user clicks a map marker or sidebar card.
- `clearSelection()`: sets `selectedThread` to null

**Action handlers (return functions for components to call):**
- `handleDeleteThread(threadId: number)`: calls `deleteThread(threadId)`, toast success '모집글이 삭제되었습니다.', clear selection, re-fetch
- `startEdit(threadId: number)`: sets `editingThreadId`, switches to RECRUIT tab
- No join/apply handler here -- that goes in Plan 03's component rewire

**NO polling.** Remove ALL setInterval for data fetch. Only the 60s expiry timer remains.
**NO old imports.** Do not import from `services/api/threadService`, `services/api/memberService`, `services/api/locationService`, or `ThreadType` from `types/index.ts`.

Import from: `@/api/threads` (specifically: `getThreads`, `getThreadMap`, `getHotspots`, `getThread`, `deleteThread`), `@/api/pets` (`getMyPets`), `@/store/useConfigStore`, `sonner` (`toast`), `react`.
