**Form fields layout (single page, not sections -- Claude's discretion):**

1. **Title** -- text input, max 30 chars, counter showing `{form.title.length}/30`, required
2. **Walk Date** -- date input (`type="date"`), min today, required
3. **Start Time** -- time input (`type="time"`), required. On change, compose ISO datetime: `${form.walkDate}T${timeValue}:00`
4. **End Time** -- time input (`type="time"`), optional. Same ISO composition.
5. **Description** -- textarea, max 500 chars, counter showing `{form.description.length}/500`
6. **Chat Type** -- toggle switch between INDIVIDUAL and GROUP:
   - INDIVIDUAL (default): "1:1 채팅" with tooltip "참여자와 개별 채팅방이 생성됩니다"

- `form.startTime !== ''`
