<objective>
Establish the crash-prevention infrastructure for the entire frontend: API proxy to backend, MSW toggle fix, global error boundary, per-route error boundaries, and a reusable section-level ErrorBoundary component.

Purpose: Without this infrastructure, (a) all API calls fail because there is no proxy to the backend, (b) MSW cannot be disabled for real-backend testing, and (c) any unhandled exception crashes the entire page with a white screen. This plan creates the safety net that catches all crashes.

Output: next.config.ts with rewrites, fixed MSWProvider, global-error.tsx, error.tsx for every route, reusable ErrorBoundary component.
</objective>
