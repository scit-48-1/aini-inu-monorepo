**page.tsx -- fetchThreads with location:**
- Use `navigator.geolocation.getCurrentPosition` to get user coordinates before fetching threads
- Wrap in a helper or inline: try geolocation first, fall back to Seoul City Hall coordinates (37.5666, 126.9784) if unavailable
- Pass `latitude`, `longitude`, `radius: 5` (5km default) to `getThreads()`:
  ```typescript
  const fetchThreads = useCallback(async () => {
    setThreads({ status: 'loading' });
    try {
      // Try to get user location, fallback to Seoul City Hall
      let lat = 37.5666;
      let lng = 126.9784;
      try {
        const pos = await new Promise<GeolocationPosition>((resolve, reject) =>
          navigator.geolocation.getCurrentPosition(resolve, reject, { timeout: 5000 })
        );
        lat = pos.coords.latitude;
        lng = pos.coords.longitude;
      } catch {
        // Use default Seoul coordinates
      }
      const res = await getThreads({ page: 0, size: 3, latitude: lat, longitude: lng, radius: 5 });
      if (res.content.length === 0) {
        setThreads({ status: 'empty' });
      } else {
        setThreads({ status: 'success', data: res.content });
      }
    } catch {
      setThreads({ status: 'error', message: '동네 소식을 불러오지 못했습니다.' });
    }
  }, []);
  ```

**around-me/page.tsx -- read threadId and auto-select:**
- Import `useSearchParams` from `next/navigation`
- Import `getThread` from `@/api/threads`
- Add `const searchParams = useSearchParams()` inside the component
- Add a `useEffect` that reads `searchParams.get('threadId')`, and if present:
  1. Calls `getThread(Number(threadId))` to fetch the full ThreadResponse
  2. Calls `selectThread(thread)` to select it on the map
  3. Only runs once on mount (use a ref to prevent re-runs)
  ```typescript
  const searchParams = useSearchParams();
  const threadIdHandled = useRef(false);

  useEffect(() => {
    const threadIdParam = searchParams.get('threadId');
    if (!threadIdParam || threadIdHandled.current) return;
    threadIdHandled.current = true;
    const fetchAndSelect = async () => {
      try {
        const thread = await getThread(Number(threadIdParam));
        selectThread(thread);
      } catch {
        // Thread may not exist anymore, ignore
      }
    };
    fetchAndSelect();
  }, [searchParams, selectThread]);
  ```
